A Tebex-integrated donator shop for FiveM. Players buy currency packages from your Tebex store, redeem the order in-game, and spend the currency on vehicles, items, weapons, and limited-time deals. You manage the catalogue from an in-game admin panel — no SQL editing, no script changes.

## What you get

* Works on **Qbox, QBCore, and ESX** — auto-detected
* **Drag-and-drop install** — database tables create themselves on first boot
* **Real Tebex integration** — orders validated against Tebex; refunds and chargebacks rejected automatically
* **In-game admin panel** — create / edit / delete packages, categories, and special deals without restarting
* **Custom currency and shop name** — call it Coins, Credits, Gems, Tokens, anything
* **Trending deals** — bundle items at a discount with a live countdown timer
* **Gift system** — players can gift coin packages to other online players
* **Test drive** — players can test a vehicle before committing
* **Live theme editor** — colours, background image, blur, currency icon, all editable in-game
* **Auto vehicle-keys integration** — qbx_vehiclekeys, qb-vehiclekeys, wasabi, MrNewb, qs, Renewed, t1ger, mono, jaksam
* **Discord webhook logs** — purchases, redemptions, and admin actions, in the channels you choose
* `escrow_ignore` on the config files so your settings survive updates
