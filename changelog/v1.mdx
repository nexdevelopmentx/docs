## v1.0.0

**Added**

* **NEX Garage** — initial public release. Multi-framework garage system with vehicle storage, transfers, impound checking, and React UI.
* **NEX Elevators** — initial public release. Premium elevator system with custom UI, in-game admin panel, and multi-framework support.
* **NEX Queue** — initial public release. Discord-driven priority queue with 8 Adaptive Card themes, role-based whitelist & bypass, optional live waitlist embed, and runtime exports for theme hot-swap and Discord lookup.
