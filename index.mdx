Welcome to the **NEX Development** documentation hub.

This is where you'll find install instructions, configuration guides, and troubleshooting for all of our FiveM scripts. Use the sidebar to jump to a specific script.

## What you'll find here

* **General** — getting started, FAQ, and how to reach support.
* **NEX Garage** — public garage system with multi-framework support.
* **NEX Elevators** — in-game elevator system with a live admin panel.
* **NEX DJ Booth** — YouTube-powered DJ booths with positional audio.
* **NEX Hotels** — full hotel system with rentals, storage, and recovery.
* **NEX UI Pack** — drop-in UI skin for `ox_lib`.
* **NEX Queue** — Discord priority queue with 8 card themes, whitelist & live waitlist embed.
* **NEX Crafting** — multi-framework crafting system with an in-game creator panel, XP, and access rules.
* **NEX Item Whitelist** — lock `ox_inventory` items down by player or job, managed in-game.
* **NEX Dealerships** — multi-framework vehicle dealership with player-owned lots, financing, stock, employees, and a React showroom UI.
* **NEX Coords Tool** — standalone in-game coords saver: copy your current position as JSON, Lua table, vector3, or vector4 with one click.
* **Changelog** — version history.
