A clean elevator system for FiveM. Players walk into an elevator, pick a floor, and get teleported with a screen fade and sound effect. You create and edit elevators entirely in-game — no config editing, no restarts.

## In this section

* [Installation](/nex-elevators/installation)
* [Configuration](/nex-elevators/configuration)
* [Admin panel](/nex-elevators/admin-panel)

## What you get

* Works on **QBCore, Qbox, ESX, and Ox Core**
* In-game admin panel — create, edit, and delete elevators live
* "Grab coords" tool — walk to a floor, press E, done
* Per-elevator and per-floor job restrictions (with minimum grade)
* Smooth screen-fade teleport with sound effects
* No database needed — data is stored in a local JSON file
